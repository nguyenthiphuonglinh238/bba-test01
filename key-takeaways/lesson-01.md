4. Kiến thức cài đặt 
- NVM: node version manager (Dễ chuyển đổi nhiều phiên bản nodejs)
- git: quản lý source code
- github: chia sẻ code, làm việc nhóm
- Cấu hính mặc định:
+ git config --global user.name"<tên bạn>"
+ git config --global user.email"<email của bạn>"
+ git config --global init.defaultBranch.main
- SHH key:
+ id_rsa (giữ bí mật) & id_rsa.pub ( có thể gửi)
+ xác thực đăng nhập dễ hơn
+ ~/.ssh (~ đại diện thư mục home)
- cat ~/.ssh/id_rsa.pub : Lấy nội dung ssh key
- npm init playwright@latest: cái đặt playwright ( nhấn enter liên tục đến khi gặp happy hacking)
- mấy câu lệnh đưa code lên github: tạo repo (note bên lesson 2)
- git remote add origin <ssh_link>: Liên kết repo vừa tạo vs git
